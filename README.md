# 3-SAT-solver-algorithm
## Алгоритм  решения задач выполнимости булевой формулы 3-SAT с заранее определенной сложностью

Простой алгоритм, позволяющий решить любую задачу, представленную в виде 3-SAT выражения<br/>
Сложность в худшем случае O(2^(n/2))
Сложность в лучшем случае O(1)

Вход: Дана функция F от n переменных, представленная в виде 3-CNF выражения

1. Создадим пустой массив переменных P (множество P). 
2. В выражении выберем переменную xi такую, что бы она входила в максимальное число клозов (дизъюнктов). Запишем эту переменную в массив P. Разделим функцию на две части - в левой части оставим клозы, в которые xi не входит, в правую перенесём все клозы, в которые xi входит. Будем выполнять пункт 2 на левой части функции до тех пор, пока в ней не останется клозов. 
3. Утверждение: перебор переменных из множества P всегда обращает исходную функцию в 2-CNF выражение, которое решается за линейное время.

Доказательство:

![image](https://github.com/user-attachments/assets/59e30b0b-b6b4-43f8-8dff-8506a1138711)

На представленном выше рисунке видно, что при подстановке всех значений в массив P, исходная функция F обращается в 2-CNF выражение.
Перебирая все варианты назначений {P} переменных из массива P, мы на каждом шаге получим 2-CNF выражение. Перебрав все варианты назначения P и проверив разрешимость каждого из 2-CNF выражения, мы получим все возможные варианты решений исходной F.

4. Полный перебор переменных множества P гарантированно находит решение F. Сложность O(2^|P|)

5. Определим множество Q как все переменные из первоначальной функции F, которые не входят в P. Q=N\P

Для Q будем перебирать все возможные назначения переменных {Q}
Для каждого назначения будем использовать обратное распространение значений переменных из {Q} на P
При этом, так как лексографически все переменные из Q находятся правее переменных из P, мы получаем направленный ацикличный граф, проходя который в обратном направлении (от разрешенных переменных к неразрешенным), мы гарантированно получим присвоение для всех переменных P. Для случаев, когда обе 2-CNF формулы для литералов какой-то переменной y \(\in \) P мы получаем, что результат выполним при любом значении y, мы можем выбрать любое из этих значений для дальнейшего распространения в графе, так как это не влияет на выполнимость (TODO: нужна отдельныя страница с обоснованием, обоснование есть, страницы нет)

6. Алгоритм перебора переменных из множества Q гарантированно находит решение F. Сложность O(2^|Q|).

Вывод: Алгоритм эффективно и детерминированно решает любую 3-SAT задачу от n переменных, представленную в виде 3-CNF. Сложность (верхняя оценка) O(2^(min(|P|,|Q|)). Так как |P|+|Q|=n, Общая верхняя граница сложности алгоритма оценивается как O(2^(n/2))

Побочные эффекты данного метода: Мы за полиноминальное время poly(n) можем обнаруживать переменные, фиктивные при данном лексографическом порядке разложения.
